# Salaam Gem

By [Ali El-Ashram](http://aliel.herokuapp.com)

## Description

Practice making my first gem.

## Information

    Greet the world with Peace!

    Example:
       >> Salaam.hi("arabic")
       => Salaam 'alayka ya RasulAllah

     Arguments:
       language: (String)

## Installation

gem install salaam

Gemfile: gem "salaam", "~> 0.0.3"
